# AgriSense Crop Disease Detection

## Overview
AgriSense is a machine learning project that detects crop diseases from leaf images using deep learning techniques.

## Dataset
The project uses the PlantVillage dataset containing healthy and diseased crop leaf images.

## Features
- Crop disease detection
- Image preprocessing
- CNN model training
- Disease prediction
- Model evaluation

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- OpenCV

## Project Structure
- cnn_training.py
- disease_prediction.py
- image_preprocessing.py
- model_evaluation.py
